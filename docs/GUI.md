## 手动扫描
打开Cobra手动扫描界面
```
http://127.0.0.1:5000 （此处域名和端口根据你的config文件而定）
然后填入你的项目git地址，你可以用这个地址做演示https://github.com/ethicalhack3r/DVWA.git
```

![Cobra手动扫描界面](https://raw.githubusercontent.com/wufeifei/cobra/master/docs/MANUAL.png)

## 扫描报告
